Aplicación full-stack para Alkemy

Esta página fue realizada para completar el challenge de ingreso a Alkemy en el cual se nos solicitó que hagamos un front que haga manejo de JWT mientras consume una API rest que nosotros debíamos desarrollar. El backend fue hecho con Express, JWT, Bcrypt y Sequelize consumiendo una base de datos PostgreSQL. Recientemente le agregue un par de funciones para utilizar en mis otros sitios haciendo peticiones a un endpoint aparte. La página esta hosteada en Github pages, el back esta hosteado en Railway, el repositorio se puede ver acá

Trabajo Freelance para Parient SRL

Este es mi primer trabajo freelance, en esta ocasión, una fábrica de tarugos e insumos descartables necesitaba una landing básica que incluyera un catalogo, lo llevamos al siguiente nivel armando un sistema donde los clientes pudieran armar su presupuesto de manera interactiva y solicitarlo a traves de WhatsApp. Este trabajo fue hecho con Javascript y las solicitudes de contacto son procesadas por un servidor mio hecho en node.js. El cliente quedo satisfecho y quedamos en hacer otro proyecto en el futuro :)

Calculadora de intereses compuestos
